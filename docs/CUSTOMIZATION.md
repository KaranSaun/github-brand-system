# Customization Guide

## How to Update the Banner SVG

Edit `assets/banner.svg` — the text, colors, and layout are all inline.

## How to Add a New Project

1. Add a project card to the Flagship Projects table in README.md
2. Create a project icon if needed (`assets/icons/<project>.svg`)
3. Add the project badge to the badge strip
4. Update the Current Mission section

## How to Update Colors

Edit `brand/DESIGN_SYSTEM.md` for the reference palette.
Update badge colors and SVG gradients to match.

## How to Deploy Changes

All changes to `KaranSaun/KaranSaun` are live immediately on push.
