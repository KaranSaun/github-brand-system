# Maintenance Guide

## Monthly Tasks

1. **Update pinned repos** — Ensure top 3 repos are still accurate
2. **Check badge URLs** — Verify no broken shields.io or GitHub badges
3. **Archive inactive repos** — New empty repos should be archived
4. **Update "Current Mission"** — Reflect active development

## Quarterly Tasks

1. **Review the brand guide** — Does it still represent your direction?
2. **Update roadmap** — Adjust based on actual progress
3. **Clean up GitHub Actions** — Remove unused workflows

## Annual Tasks

1. **Full profile audit** — Re-run the audit process
2. **Update banner SVG** — Refresh visual identity
3. **Update tech stack** — Add new technologies, remove unused
