# Design System

## Color Palette

### Primary
- `#0A0E17` — Deep void (background primary)
- `#0F1420` — Surface (background secondary)
- `#151C2C` — Elevated (card background)

### Accent
- `#00D4FF` — Cyan (primary accent)
- `#8B5CF6` — Purple (secondary accent)
- `#10B981` — Green (success / positive)
- `#EF4444` — Red (error / negative)

### Text
- `#F0F4F8` — Primary text
- `#94A3B8` — Secondary text
- `#64748B` — Tertiary text
- `#475569` — Muted text

### Borders
- `rgba(148, 163, 184, 0.08)` — Subtle border
- `rgba(148, 163, 184, 0.12)` — Default border
- `rgba(0, 212, 255, 0.3)` — Active border

## Typography

- **Headings:** Inter (600/700 weight)
- **Body:** Inter (400 weight)
- **Code/Monospace:** JetBrains Mono

### Scale
- Hero: 48px / 3rem
- H1: 32px / 2rem
- H2: 24px / 1.5rem
- H3: 20px / 1.25rem
- Body: 16px / 1rem
- Small: 14px / 0.875rem
- Micro: 12px / 0.75rem

## Spacing

- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- 2xl: 48px
- 3xl: 64px

## Border Radius

- Card: 16px
- Button: 12px
- Badge: 8px
- Input: 10px

## Glow Effects

- Cyan glow: `0 0 20px rgba(0, 212, 255, 0.15), 0 0 60px rgba(0, 212, 255, 0.05)`
- Purple glow: `0 0 20px rgba(139, 92, 246, 0.15), 0 0 60px rgba(139, 92, 246, 0.05)`

## Glassmorphism

- Background: `rgba(15, 20, 32, 0.7)`
- Blur: `blur(20px) saturate(1.4)`
- Border: `1px solid rgba(148, 163, 184, 0.08)`
